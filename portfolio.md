description : Developpement d'un jeu de type R-Type de A-Z from scratch, en trois parties ; ECS qui est un design pattern etant la base d'un game engine pour "construire un jeu" a l'aide de bloc. Un client graphique permettant d'afficher le jeu est d'y jouer. Un Serveur permettant la communication entre les clients et source de verite du jeu. Le tout est multi-joueur avec notre propre RFC, par "room" de jeu, multi-threaded et multi-plateforme. Developper en collaboration avec dragusheen.com et Hardeol, deferdam, Aeliondw sur github.

technologie : C++, SDL, Network & Multi-thread
